# Support

## Getting Help

- **Bug reports** — [GitHub Issues](https://github.com/groupesti/laravel-ca-pkcs12/issues) (bugs only, please include PHP/Laravel versions and a minimal reproduction).
- **Questions & discussions** — [GitHub Discussions](https://github.com/groupesti/laravel-ca-pkcs12/discussions).
- **Security vulnerabilities** — See [SECURITY.md](SECURITY.md). Do **not** open a public issue.

## Before Opening an Issue

Please include the following in every bug report:

- Package version (`composer show groupesti/laravel-ca-pkcs12`)
- PHP version (`php -v`) — must be 8.4+
- Laravel version (`php artisan --version`) — 12.x or 13.x
- Steps to reproduce the issue
- Expected vs. actual behavior

## Important

GitHub Issues are **not** a general support forum. Questions and how-to requests should be posted in GitHub Discussions.
